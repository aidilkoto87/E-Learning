Repository ini merupakan project perkuliahan yang dibuat sebagai bagian dari tugas akademik pada mata kuliah terkait pengembangan sistem informasi dan pemrograman web menggunakan framework Laravel. Project ini bertujuan untuk menerapkan konsep CRUD, manajemen database, autentikasi pengguna, serta pengelolaan sistem berbasis web secara terstruktur.

Pengembangan project ini dilakukan untuk memenuhi kebutuhan pembelajaran sekaligus meningkatkan pemahaman dalam implementasi teknologi web modern, khususnya dalam penggunaan framework Laravel, integrasi database, dan penerapan logika sistem yang efisien.

Repository ini digunakan sebagai dokumentasi hasil pembelajaran, pengembangan tugas kuliah, serta portofolio akademik.
